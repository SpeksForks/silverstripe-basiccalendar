Basic Events Module
==================

## Author
* Andrew Houle
* http://andyhoule.com

## Requirements
* SilverStripe minimum version 3.1+.

## Installation
* Add the news folder to your site's root.
* Run http://yoursite.com/dev/build/?flush=all
* Create a new Calendar page and add Calendar Entries.
